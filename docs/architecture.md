# 系统架构文档

## 1. 技术栈
- 前端：HTML + CSS + JavaScript
- 后端：Python（Flask 框架）
- 数据存储：本地文本文件 / SQLite 数据库

## 2. 整体架构
采用前后端分离架构：
1.  前端层：负责用户界面展示、接收用户操作
2.  业务逻辑层：处理核心业务流程（如数据校验、统计计算）
3.  数据层：负责数据的存储与读取

## 3. 模块划分
- 用户模块：登录、权限校验
- 数据管理模块：数据的增删改查、导入导出
- 统计模块：数据计算与结果展示

## 4. 核心流程
用户登录 → 选择功能 → 提交操作请求 → 后端处理 → 返回结果并展示
