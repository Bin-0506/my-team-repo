项目需求文档

1. 项目背景
本项目为课程小组作业，旨在开发一个简易的文章开发
 2. 功能需求
- 用户登录：支持账号密码登录，区分管理员与普通用户权限
- 数据录入：支持手动添加、修改、删除数据
- 数据查询：支持按关键词、条件筛选查询
- 数据统计：自动生成基础统计结果（如总数、平均值）
- 数据导出：支持将数据导出为文本文件
 3. 非功能需求
- 易用性：界面简洁，操作流程清晰
- 兼容性：支持主流浏览器（Chrome、Edge）
- 稳定性：核心功能运行稳定，无致命错误
4. 验收标准
- 所有核心功能可正常使用
- 无严重bug，能完成基础业务流程
- 文档齐全，代码可正常运行
